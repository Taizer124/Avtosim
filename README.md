# City Racer

**City Racer** — автосимулятор на Unity с упором на реалистичное вождение, поддержку игровых рулей и VR. Игрок катается по открытому городу, покупает и прокачивает машины за заработанные монеты и участвует в гонках против ИИ-соперников.

Проект также используется как учебная площадка для работы с симрейсинг-оборудованием (руль + педали + VR-шлем).

## 🎮 Основные механики

- **Реалистичная физика вождения** — управляемая модель автомобиля с настраиваемыми узлами (двигатель, коробка, подвеска, тормоза, шины, наддув).
- **Три типа трансмиссии:**
  - *Автомат* — передачи переключаются сами;
  - *Секвентальная* — переключение вверх/вниз паддлами;
  - *Механика (H-pattern)* — ручной выбор передачи со сцеплением (передача включается только при выжатом сцеплении).
- **Типы привода:** задний (RWD), полный (AWD), передний (FWD).
- **Выбор и покупка машин** — каждая машина имеет цену; за монеты открываются новые авто. Баланс и покупки сохраняются между сессиями.
- **Экономика и прогрессия** — за призовые места в гонке начисляются монеты (награды настраиваются по местам).
- **Гонки с ИИ** — старт/финиш зоны, круги, таблица лидеров, соперники на трассе.
- **Свободная езда по городу** — режим «песочницы» с возможностью на ходу менять коробку, привод и пресеты настроек.
- **Приборная панель** — спидометр и тахометр (стрелочные, заполняющиеся или физический 3D-объект), руль в кабине повторяет поворот реального руля.

## 🕹️ Управление и оборудование

- **Игровые рули:** Logitech G29 и MOZA (через официальный MOZA SDK) — руль, педали газ/тормоз/сцепление, кнопки и шифтер.
- **Клавиатура:** WASD — газ/тормоз/руль, F — сцепление, 1–7 — передачи механики, и т.д.
- **Мультиввод** — клавиатура и руль работают одновременно, устройства взаимозаменяемы.
- **VR** — поддержка шлема (XR), вид из кабины; для ПК доступен обзор камерой.

## 🛠️ Технологии

- **Движок:** Unity (Universal Render Pipeline)
- **Ввод:** Unity Input System (единая раскладка для руля/клавиатуры/MOZA)
- **VR:** XR Interaction Toolkit
- **Оборудование:** Logitech Steering Wheel SDK, MOZA API SDK
